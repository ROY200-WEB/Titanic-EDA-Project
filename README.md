# Titanic Exploratory Data Analysis (EDA)

This project performs Exploratory Data Analysis on the Titanic dataset using Python libraries like Pandas, Matplotlib, and Seaborn.

## Skills Used
- Data Cleaning
- Data Visualization
- Descriptive Statistics
- Trend Analysis
- Outlier Detection

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
